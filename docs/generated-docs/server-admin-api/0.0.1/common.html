회원 조회

POST /api/members/{id}

@title 회원 조회.

Example request

POST /cjconnect/admin/api/api/members/1 HTTP/1.1
Content-Type: application/json;charset=UTF-8
Accept: application/json
Content-Length: 14
Host: api-admin-stage.unban.ai

{
  "id" : 1
}

Query parameters

No parameters.

id

Integer

true
